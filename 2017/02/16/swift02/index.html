Swift3-02 初见

by · 2017年02月16日 · 224 Words · ~1min reading time | Improve on

首先我们来在屏幕上输出一个 hello world

print("hello world")

不需要 import 任何库,可以直接在全局作用域使用。代码结尾不要带分号。

Xcode 提供了代码预览功能 Playground . 使用它可以加快 swift的学习速度

初始化一个变量和常量 var a = 2 //可变数据 a = 3 let b = 2 //不可变

if else 雨晶 if a == 2{ print(“true”) } else{ print(“false”) }

for in 遍历语句(此外还有 ,repeat..while ,for 循环3.0中已经被弃用)

for index in 0...100{
   var c = Double(index)/20.0
}

while

while 1 == 1 {
    a += 1
    if a == 100{
        break
    }
    print("do something",a)

}

repeat… while

repeat
{
    a += 1
    if a > 100{
        break
    }
    print("do something",a)

}while( 1 == 1 );

断言

let age = -3 
assert(age >= 0, "A person's age cannot be less than zero") 
// 因为 age < 0，所以断言会触发
