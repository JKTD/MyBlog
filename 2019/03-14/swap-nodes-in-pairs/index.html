两两交换链表中的节点 Swap Nodes in Pairs

Thu, Mar 14, 2019
Description

给定一个链表，两两交换其中相邻的节点，并返回交换后的链表。

你不能只是单纯的改变节点内部的值，而是需要实际的进行节点交换。

Example
给定 1->2->3->4, 
返回 2->1->4->3.
Solution
type ListNode struct {
    Val int
    Next *ListNode
}


func swapPairs(head *ListNode) *ListNode {
    if head == nil || head.Next == nil {
        return head
    }
    next := head.Next
    head.Next = swapPairs(next.Next)
    next.Next = head
    return next
}
