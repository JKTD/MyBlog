go bin执行问题记录

Sun, Feb 24, 2019

因为${GOPATH}/bin目录下的执行文件无法被中断直接执行 需要Source之后才能使用， 只是在 .bash_profile文件里如此设置 需要Source之后才能被命令行识别

export GOBIN=/Users/ori/go/bin
export PATH=$PATH:$GOBIN

export PUB_HOSTED_URL=https://pub.flutter-io.cn
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
export  PATH=~/code/flutter/flutter/bin:$PATH

最好的处理方式还是直接把${GOPATH}/bin目录下的执行文件移动到go的Root的bin目录中更好

mv $GOPATH/bin/** /usr/local/go/bin
