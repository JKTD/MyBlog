回文数

Sun, Mar 10, 2019
Description

判断一个整数是否是回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

Example
输入: 121
输出: true


输入: -121
输出: false
解释: 从左向右读, 为 -121 。 从右向左读, 为 121- 。因此它不是一个回文数。

输入: 10
输出: false
解释: 从右向左读, 为 01 。因此它不是一个回文数。
Solution
func isPalindrome(x int) bool {
    if x < 0  {
        return false
    }
    if x %10 == 0 && x != 0 {
        return false
    }
    result := 0

    for x > result {
        result = result*10 + x%10
        x = x/10
    }
    if result == x || x == result/10{
         return true
    }
    return false
}
